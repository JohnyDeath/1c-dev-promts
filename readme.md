# 1С:Предприятие — AI-инструменты для разработки

Набор инструментов для интеграции ИИ-ассистентов с платформой 1С:Предприятие 8.

## Структура проекта

```
├── powers/1c-mcp/          # Power для Kiro IDE
├── .kilocode/skills/1c-mcp/ # Skill для Kilocode
└── promts/                  # Промты для разработки на 1С
```

## Powers для Kiro

**Каталог:** `powers/1c-mcp/`

Power для IDE Kiro, обеспечивающий интеграцию с базами данных 1С через MCP-сервер.

Использует MCP-сервер: [RooLee10/1c-mcp-tools](https://github.com/RooLee10/1c-mcp-tools)

**Возможности:**
- Выполнение запросов к базе данных 1С
- Исследование структуры метаданных конфигурации
- Получение описаний объектов (реквизиты, табличные части)
- Работа с навигационными ссылками 1С

**Настройка:** отредактируйте `powers/1c-mcp/mcp.json`, указав URL вашего HTTP-сервиса 1С и учётные данные.

## Skills для Kilocode

**Каталог:** `.kilocode/skills/1c-mcp/`

Тот же функционал, что и Power для Kiro, но в формате навыка для расширения Kilocode.

## Промты для разработки

**Каталог:** `promts/`

- `1cdev.md` — руководство по разработке на платформе 1С:Предприятие 8 для ИИ-ассистентов

Содержит правила кодирования, архитектуру проектов, синтаксис языка запросов и критические требования к разработке.
